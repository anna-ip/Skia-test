# React Native Skia test

An app trying some of the [React Native Skia](https://shopify.github.io/react-native-skia/) components like:
* Canvas
* Polygons
* Paths
* SVGs
* Ellipses

### Technologies
* React Native
* React Native Skia
* React Navigation
* React Native Gesture Handler
* Expo

### Run the app in terminal:
npx expo start
